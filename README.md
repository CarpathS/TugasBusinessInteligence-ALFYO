Proyek ini menampilkan analisis pasar kerja bidang Data Science. Dashboard membantu Anda melihat tren pekerjaan, kebutuhan skill, dan profil perusahaan. File dibuat dengan Power BI.

Fitur Utama
- Dashboard Jobs. Menampilkan jumlah lowongan, lokasi, level pekerjaan, dan tren permintaan.
- Dashboard Skills. Menampilkan skill yang paling banyak diminta. Menunjukkan hubungan skill dengan gaji dan level pekerjaan.
- Dashboard Company. Menampilkan daftar perusahaan perekrut. Menunjukkan ukuran perusahaan dan distribusi lowongan.
- Analisis Tambahan. Job level, job titles, salary, skill likelihood, pengalaman.

Data
- Data berasal dari studi kasus Analysing Job Market Data.
- Format awal menggunakan CSV.
- Data dibersihkan lalu dimuat ke Power BI.
- Transformasi mencakup penghapusan duplikat dan standarisasi nama kolom.

Tujuan
- Membantu Anda memahami kondisi pasar kerja Data Science.
- Memberi wawasan skill yang perlu Anda kuasai.
- Membantu perusahaan memetakan kebutuhan talenta.

Navigasi Dashboard
- Home. Menu utama dengan tiga kategori.
- Jobs. Analisis lowongan.
- Skills. Analisis skill.
- Company. Analisis perusahaan.
- Lembar lain berisi analisis lanjutan.
